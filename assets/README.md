## Panduan Kontribusi Gambar

Kalau kamu ingin mengganti atau menambah gambar:

1. Gunakan format PNG, resolusi minimal 1200px lebar
2. Nama file deskriptif dengan tanda hubung (contoh: `nama-gambar-deskriptif.png`)
3. Gaya visual: clean, flat illustration, palet netral — konsisten dengan gambar yang sudah ada
4. Submit melalui pull request dengan deskripsi singkat perubahan yang dibuat

Lihat [CONTRIBUTING.md](../CONTRIBUTING.md) untuk panduan umum kontribusi.
