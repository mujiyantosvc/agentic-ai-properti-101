# Prompts: Marketing Properti

Prompt untuk membuat konten marketing properti yang efektif — dari caption media sosial sampai artikel SEO dan follow-up script.

---

## Prompt 1: Caption Instagram (3 Variasi)

```
Buat 3 variasi caption Instagram untuk listing properti berikut:

Data properti:
- Tipe: [rumah tapak/apartemen/ruko/kavling]
- Spesifikasi: [LT/LB, jumlah kamar, kamar mandi, garasi]
- Lokasi: [kawasan, kota, keunggulan lokasi]
- Harga: [Rp X / "hubungi agen untuk harga"]
- Keunggulan spesifik: [misal: dekat stasiun, renovasi baru, view bagus, dll.]
- Target pembeli: [pasangan muda / keluarga / investor / ekspatriat / dll.]

Buat 3 variasi caption dengan tone berbeda:
1. **Emosional** — bayangkan kehidupan di sini, hubungan dengan keluarga
2. **Informatif** — fakta dan keunggulan konkret
3. **Story** — cerita pendek tentang masalah pembeli yang dijawab properti ini

Untuk masing-masing:
- Panjang: 150-200 kata
- Ada hook kuat di kalimat pertama
- Ada soft CTA di akhir (bukan "hubungi sekarang!" yang hard-selling)
- Sertakan 5-8 hashtag yang relevan

Tidak perlu: emoji berlebihan, kalimat klise seperti "hunian impian", angka yang belum diverifikasi.
```

## Prompt 2: Script TikTok / Reels (30 Detik)

```
Buat script video pendek 30 detik untuk properti ini:

Properti:
- [Deskripsi singkat: tipe, lokasi, keunggulan utama]
- Format yang diinginkan: [tur singkat / talking head / before-after / testimonial style]
- Persona presenter: [agen / pemilik / narator off-screen]

Struktur script (per segmen waktu):
- 0-3 detik: HOOK — sesuatu yang bikin orang berhenti scroll
- 3-10 detik: MASALAH atau PERTANYAAN yang relate dengan target audiens
- 10-20 detik: SOLUSI / penampakan properti yang menjawab masalah itu
- 20-27 detik: DETAIL KUNCI 2-3 fakta penting
- 27-30 detik: CTA RINGAN

Target audiens: [X]
Platform: [TikTok / Instagram Reels / YouTube Shorts]

Format output:
- Narasi per detik
- Note untuk visual/b-roll yang cocok di tiap segmen
- Alternatif hook (2 opsi)
```

## Prompt 3: Deskripsi Listing Properti

```
Buat deskripsi listing untuk properti berikut:

Data properti (semua wajib diisi, jangan ada yang dikira-kira):
- Tipe: [X]
- Luas tanah: [X m²], luas bangunan: [X m²]
- Jumlah lantai: [X]
- Kamar tidur: [X], kamar mandi: [X]
- Fasilitas: [sebutkan semua: garasi, taman, carport, dll.]
- Kondisi: [baru / second / furnish / semi furnish / kosong]
- Keunggulan lokasi: [akses ke mana, jarak ke fasilitas penting]
- Status legalitas: [SHM / HGB / sudah IMB] (harus akurat)
- Harga: [Rp X] atau [rentang]
- Contact agen: [jika mau disertakan]

Target pembeli:
- [deskripsikan profil pembeli yang paling mungkin untuk properti ini]

Format:
- Panjang: 200-250 kata
- Struktur: keunggulan utama → spesifikasi → lokasi → aksi
- Tone: profesional tapi hangat, tidak hard-selling
- Sertakan: satu kalimat tentang kemungkinan "kehidupan di sini"
- Hindari: klaim "terbaik", "termurah", "terlengkap" tanpa data

PENTING: Jangan mengarang fakta. Kalau ada data yang tidak saya berikan,
tanyakan sebelum menulis, atau tandai dengan [PERLU DIKONFIRMASI].
```

## Prompt 4: Landing Page Properti (1 Halaman)

```
Buat teks landing page singkat untuk properti yang sedang aktif dipasarkan.

Properti:
- [Data lengkap seperti di atas]

Struktur landing page yang diinginkan:

1. HEADLINE — satu kalimat yang merangkum proposisi utama
   Bukan: "Rumah Dijual di Jakarta Selatan"
   Tapi: sesuatu yang langsung relevan dengan masalah target pembeli

2. SUBHEADLINE — 1-2 kalimat pendukung headline

3. KEUNGGULAN UTAMA — 3 poin bullet, singkat dan konkret

4. DESKRIPSI SINGKAT — 2-3 paragraf, 100-150 kata

5. SPESIFIKASI — tabel atau bullet poin data teknis

6. LOKASI DAN AKSES — highlight jarak ke 3-5 fasilitas penting

7. FAQ CEPAT — 3 pertanyaan yang paling sering ditanyakan calon pembeli

8. CTA SECTION — ajakan untuk menghubungi/jadwal kunjungan
   Tone: natural, tidak memaksa

Target: [profil pembeli]
```

## Prompt 5: Artikel SEO Properti

```
Buat outline artikel blog SEO-friendly tentang properti.

Topik: [misal: "Panduan Beli Rumah Pertama di [kota] dengan KPR"]
Target pembaca: [misal: pasangan muda yang baru menikah, berpenghasilan 8-15 juta/bulan]
Tujuan artikel: [edukasi / awareness untuk brand agen / traffic organik]

Outline yang diinginkan:
- H1: Judul utama (sertakan keyword target)
- Meta description: 150-160 karakter
- H2 dan H3: struktur artikel dengan topik per bagian
- Panjang total: [1000-1500 / 1500-2500] kata
- Jumlah referensi internal (link ke halaman lain) yang disarankan

Konten yang harus ada:
- Informasi yang genuinely membantu pembaca mengambil keputusan
- Data atau angka yang konkret (bukan estimasi ngawur)
- Bagian yang menjawab pertanyaan umum calon pembeli
- CTA yang natural di akhir, bukan di tengah paragraf yang mengganggu

Yang harus dihindari:
- Keyword stuffing
- Klaim tanpa bukti
- Teks yang terasa seperti ditulis untuk robot, bukan manusia
```

## Prompt 6: Iklan Soft Selling

```
Buat teks iklan soft-selling untuk properti yang cocok digunakan di:
[Instagram / Facebook / Google Display / WhatsApp Broadcast]

Prinsip soft-selling yang harus diikuti:
- Mulai dari masalah atau kebutuhan pembeli, bukan dari keunggulan produk
- Tampilkan properti sebagai solusi, bukan sebagai objek jual
- Tidak ada kata: "segera", "terbatas", "jangan sampai menyesal", "kesempatan langka"
- Ada informasi yang berguna — bukan hanya promosi kosong

Data properti: [X]
Target audiens: [X]
Budget harga: [Rp X — atau range yang nyaman disebutkan]

Output:
1. Teks iklan versi pendek (50 kata) — untuk display ads
2. Teks iklan versi menengah (100-120 kata) — untuk feed Instagram
3. Teks iklan versi WA broadcast (informal, personal) — untuk list kontak existing

Sertakan: hook opening + value proposition + soft CTA untuk masing-masing.
```

## Prompt 7: Konten Edukasi Properti

```
Buat konten edukasi properti yang secara tidak langsung membangun trust
dan menunjukkan keahlian agen/developer.

Topik edukasi: [pilih atau buat yang relevan]
- "Biaya-biaya yang sering lupa dihitung saat beli rumah"
- "Perbedaan KPR subsidi dan non-subsidi"  
- "Cara baca sertifikat tanah sebelum beli"
- "Hal yang wajib dicek saat kunjungan rumah second"
- "Kapan waktu yang tepat investasi di properti?"
- "Renovasi atau beli baru — pertimbangannya apa?"

Format: [carousel Instagram (6-8 slide) / artikel pendek 500 kata / thread Twitter/X / 
video script 60 detik]

Tone:
- Informatif, seperti teman yang lebih tahu berbagi pengalaman
- Tidak menggurui atau terlalu akademik
- Ada 1-2 poin yang sedikit counter-intuitive atau mengejutkan

Di akhir konten (bukan di tengah): 1 kalimat soft tentang bagaimana agen ini bisa membantu
```
